# 🌾 FarmDesk

**FarmDesk** is a comprehensive farm management platform designed to help farmers and agricultural professionals streamline operations, track resources, and optimize farm productivity.

## 📋 Project Overview

FarmDesk provides a modern, intuitive interface for managing farm operations including:
- Resource tracking and inventory management
- Task scheduling and workflow management
- Data visualization and reporting
- Real-time farm metrics and analytics

**Project URL**: https://lovable.dev/projects/53410996-a867-482d-8f69-ebe442833ddc

## 🚀 Getting Started

### Requirements
- Node.js & npm installed ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

### Local Development Setup

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd farmdesk

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

The app will be available with hot-reloading enabled for instant preview of your changes.

## 🛠️ Technology Stack

FarmDesk is built with modern web technologies:

- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite
- **UI Components**: shadcn/ui with Radix UI primitives
- **Styling**: Tailwind CSS
- **Forms**: React Hook Form with Zod validation
- **Database**: Supabase (PostgreSQL)
- **State Management**: TanStack Query
- **Features**: 
  - QR code scanning and generation
  - PDF generation for reports
  - Drag-and-drop interfaces (dnd-kit)
  - Progressive Web App (PWA) support
  - Charts and data visualization (Recharts)
